This folder contains the Jolly logo image.

Logo file should be named: jolly-logo.png

The logo is displayed in the sidebar header on all pages:
- home.html
- tour-analysis.html
- campaign-analysis.html
- survey-analysis.html

Logo specifications:
- Format: PNG (recommended) or SVG
- Display size: 40x40px (circular)
- The CSS automatically applies border-radius: 50% to make it circular
